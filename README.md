# API REST de Talleres - Desarrollo Web Backend

Este repositorio contiene el código y la documentación de los talleres desarrollados para la clase de **Backend**.
Esta API REST moderna está construida con Express.js, TypeScript y cuenta con documentación automática generada por Swagger.
Los endpoints están organizados por taller, y cada función implementada en un taller corresponde a un endpoint específico.

## Instalación

```bash
git clone https://github.com/santiagohoyos20/talleres-dllo-backend
cd talleres-dllo-backend

npm install
```

## Ejecución

### Modo desarrollo
```bash
npm run dev
```

### Modo producción
```bash
npm run build
npm start
```


## Documentación API

Una vez iniciado el servidor, puedes acceder a la documentación interactiva de Swagger, donde podrás explorar y probar todos los endpoints directamente desde el navegador, en la siguiente URL:

```
http://localhost:3000/api/v1/docs
```
